Prerequisites

Section titled Prerequisites

  1. An Astro project - If you don’t have an Astro project yet, our Installation guide will get you up and running in no time.

  2. A ButterCMS account. If you don’t have an account, you can sign up for a free trial.

  3. Your ButterCMS API Token - You can find your API Token on the Settings page.
