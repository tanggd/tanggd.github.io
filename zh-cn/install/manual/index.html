手动安装 Astro

准备好安装 Astro 了？跟着我们的自动化或手动设置教程来开始吧。

前提准备

Section titled 前提准备
  • Node.js - 14.15.0，v16.0.0 或更高版本。
  • 文本编辑器 - 我们建议使用安装有 Astro 官方扩展的 VS Code。
  • 终端（Terminal） - Astro 可以通过其命令行界面 (CLI) 访问。
